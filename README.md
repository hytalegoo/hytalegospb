# Hytale Fan Site

Это фанатский сайт посвященный игре Hytale. Сайт включает в себя таймеры предзаказа и релиза, информацию об игре, скриншоты, видео, раздел скачивания и popup окно, которое появляется при загрузке страницы.

## Структура Сайта

### HTML (index.html)
- **Header**: Фиксированная навигация с логотипом и меню (Главная, Об игре, Скриншоты, Видео, Скачать).
- **Hero Section**: Приветствие с описанием игры и кнопкой "Узнать больше".
- **Discord Center**: Кнопка для присоединения к Discord.
- **Info Section**: Табы с информацией об игре (Обзор, Особенности, Модинг).
- **Screenshots**: Галерея изображений с модальными окнами для просмотра.
- **Videos**: Видео плеер с трейлером Hytale.
- **Download Section**: Табы для лаунчера и прямой загрузки с формой выбора ОС.
- **Popup**: Модальное окно с изображением и текстом о релизе, появляется через 1 секунду после загрузки.

### CSS (styles.css)
- **Глобальные Стили**: Сброс, шрифты, цвета.
- **Header**: Фиксированный с градиентом и эффектами hover.
- **Hero**: Фон с градиентом и изображением, анимации.
- **Timers**: Фиксированные таймеры с пиксельным стилем и анимациями.
- **Tabs и Content**: Переключаемые секции.
- **Gallery**: Сетка изображений с hover эффектами.
- **Modal**: Полноэкранный просмотр скриншотов.
- **Forms**: Стили для выбора ОС и архитектуры.
- **Popup**: Модальное окно с анимациями bounce-in/out.
- **Responsive**: Медиа-запросы для мобильных устройств.

### JavaScript (script.js)
- **Timers**: Два таймера (предзаказ и релиз) с анимациями flip для чисел.
- **Tabs**: Переключение активных табов в info и download секциях.
- **Gallery Modal**: Открытие модального окна при клике на изображение.
- **Download Forms**: Динамическое обновление опций архитектуры в зависимости от ОС.
- **Popup**: Показ popup через 1 секунду после загрузки, закрытие по крестику или фону.
- **CTA Button**: Прокрутка к секции "Об игре".

## Функционал
- **Таймеры**: Обновляются каждую секунду, показывают время до 13 декабря 2025 (предзаказ) и 13 января 2026 (релиз).
- **Popup**: Эпичный текст с изображением, появляется автоматически.
- **Модальные Окна**: Для скриншотов.
- **Формы**: Валидация выбора ОС для скачивания.
- **Анимации**: Fade-in, flip, glow, bounce.
